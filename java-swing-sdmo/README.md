## sdmo项目
##### 简单实现登陆和注册系统：
 ##### 采用技术：
   
 - 使用Swing（GUI\CS架构）进行开发而成的可视化项目
 - 数据库采用Mysql数据库进行数据操作
 ##### 主要功能：
  
 - 可实现用户的登陆
 - 可实现新用户的注册
