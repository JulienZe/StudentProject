## java-swing-soure 项目
##### 学校教务管理系统
 ##### 采用技术：
   
 - 使用Swing（GUI\CS架构）进行开发而成的可视化项目

 ##### 主要模块：
  
 - 主模块
  1.1 登陆
  1.2 修改密码
  1.3 用户管理
  1.4 系统操作日志
  1.5 皮肤切换
  1.6 退出
 - 后台管理模块
  1.1 学生信息管理模块
  1.2 教师信息管理模块
  1.3 课程信息管理模块
  1.4 年纪信息管理模块
  1.5 班级信息管理模块
 - 课程管理模块
  1.1 选课录入
  1.2 选课结果
 - 成绩管理模块
  1.1 成绩录入
  1.2 成绩统计
  1.3 报表生成
 - 数据库管理模块
  1.1 数据初始化
  1.2 数据库备份
  1.3 数据库恢复
 - 项目说明模块
  1.1 说明
  1.2 使用教程
